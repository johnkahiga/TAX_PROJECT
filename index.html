How we calculate

  3. Deductions first: NHIF + NSSF + NHDF
  4. Taxable income = Gross − (NHIF + NSSF + NHDF)
  5. PAYE (Pay As You Earn) is calculated using progressive slabs, then personal relief (2400) is subtracted (and tax can’t go below 0).
  6. How the tax amount is produced (inside PAYE):
    • 0–24,000: Taxable × 10%
    • 24,001–32,333: 10% of 24,000 + 25% of the amount above 24,000
    • 32,334–500,000: add 30% slab for the next range
    • 500,001–800,000: add 32% slab for the next range
    • 800,001+: add 35% for the remaining amount
  7. Net salary = Gross − (NHIF + NSSF + NHDF + PAYE)
